## Тестовое задание для компании Ришат

Данный мини проект запущен тут: http://51.195.169.104:8080/

# Запуск 

### 1) Подготовка
  Переименовать файл `example.env` в `.env` и Подставить свои ключи
  Ключик для stripe можно взять тут https://dashboard.stripe.com/test/apikeys
  для джанго тут https://djecrety.ir/

### 2) Запускаем докер
  Перейти в папку проекты и выполнить:
  ```
  docker build . -t stripe-django-v0.1
  ```
  ```
  docker run -p 8080:8080 stripe-django-v0.1
  ```
  Тестовый сервер запуститься на порту 8080
  
### 3) Создаём суперпользователя
  Выполняем `docker ps`
  Смотрим имя контейнера в поле *NAME*
  Далее подставляем *NAME* в команду:
  ```
  docker exec -it NAME bash
  ```
```
  python manage.py createsuperuser
  ```
  следуем интрукциям
### 5) Заходим в админку, добавляем товары
